## DataWow Backend

This is project for DataWow NestJs Backend

## Installation

```bash
# install dependency
$ npm install
```


To run this project, you will need to add the following environment variables to your .env file

`DATABASE_URL`
`JWT_SECRET`
`PORT`
`FRONTEND_URL`


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
